# ai

## 作用
- 存放 AI 协作提示词、会话日志与输出物。

## 边界
- 仅用于协作与追踪，不作为运行时依赖。

## 文件清单
- `prompts/NEW_MODEL_HANDOFF_PROMPT.md`：模型交接提示词
- `prompts/SOP_WORKFLOW.md`：标准工作流提示词
- `prompts/UNIFIED_START_PROMPT.md`：统一启动提示词
- `sessions/YYYY-MM-DD.template.md`：会话日志模板
- `sessions/2026-04-04.md`：今日 session summary
- `sessions/2026-04-05.md`：本地预览实例与真机验收准备记录
- `outputs/`：AI 生成输出目录
