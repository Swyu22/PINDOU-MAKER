# 统一启动提示词（短版）

你现在接手本项目，请先不要直接写代码，必须按项目工作流规范执行：

【第一步：读取与对齐】
1) 读取 `CLAUDE.md`
2) 读取 `/.cloud.md`
3) 读取本次目标模块的 README / _module.md（若存在）
4) 读取目标文件及相邻关键文件的头部说明（若存在）

【第二步：用 5 条以内总结】
请输出：
- 当前目标（你理解的）
- 本次范围（会改哪些文件 / 不改哪些）
- 当前约束（组件库/架构/质量红线）
- 验收标准
- 需要回写的文档清单（至少包含 `.cloud.md`）

【第三步：执行计划】
给出本次执行步骤（含验证方式），并说明是否需要读取更多文档（spec / plan / ADR）。

【第四步：执行后强制回写】
完成后必须输出并写入：
- Last Done
- Next Actions（按优先级）
- Key Decisions（无则写无）
并更新 `.cloud.md` 与相关模块文档（如有变更）。
